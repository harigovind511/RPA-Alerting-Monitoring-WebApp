What is chatbot?

A bot is a software application that runs automated tasks(scripts) over the internet. Typically, bots perform tasks that are both simple and structurally repetitive, at a much higher rate than would be possible for a human. -Wikipedia

  • A chatbot is designed so that users can have a natural language conversation with the system to perform tasks or to get insights and these exist within chat application or as stand alone.
  • AI(NLP) component is used to give bots the ability to understand the context or intention of a user message and respond accordingly.

What are Conversational Interfaces?

  • Chat Interface:

    Users interact with the bot using text as input, can be exposed view a number of messaging applications or even on websites and SharePoint sites.

  • Voice Interface:

    User primarily interact with the bot using their voice as an input, here users can ask a question directly to the device and get their queries resolved. Bot can be easily integrated with Cortana, Alexa or even Google Assistant.

Features

  • Build in features for authentication and authorization with SSO as standard option.
  • Entire platform is built with reusability in mind, and hence most of the solutions can be scaled across geographies and application without much rework.
  • Ability to integrate with enterprise level application and tools, such as JDA, Netsuite, Pega, etc.
  • Bot can trigger RPA processes using natural language query.
